# 💪 Angel Prime

App Android personalizada de rutina de gym, creada porque me cansé de buscar ejercicios en internet cada vez que entrenaba. Tiene mis 5 días de entrenamiento, GIFs de cada ejercicio, tips de ejecución y mi plan de dieta — todo offline y en un solo lugar.

---

## ¿Por qué existe esta app?

Estaba harto de abrir el navegador, buscar el ejercicio, encontrar videos con anuncios, perder el hilo... Solo quería ver el movimiento y el tip directo. Así que la hice yo mismo.

Originalmente la construí como una **página web en HTML y CSS**, funcionaba bien pero al final del día seguía siendo una página web — tenía que abrirla en el navegador, no tenía ícono en mi pantalla, no se sentía como una app de verdad. Así que decidí convertirla a una **app Android nativa en Kotlin** para tenerla siempre a la mano como cualquier otra app.

---

## Funciones

- 📅 5 días de entrenamiento organizados (Lunes a Viernes)
- 🥗 Plan nutricional de 2,700 kcal / 130g de proteína
- 🎥 GIF animado de cada ejercicio
- 💡 Tips de ejecución en lenguaje simple ("Bebé Tips")
- 👆 Navegación con botones y swipe entre ejercicios
- 📴 100% offline, no necesita internet

## Tecnologías

- **Kotlin** — lenguaje principal
- **Android Studio** — IDE
- **Glide** — carga de GIFs animados
- **RecyclerView** — lista de días
- **Assets** — GIFs guardados localmente (offline)

## Origen

```
HTML + CSS  →  app Android nativa en Kotlin
```

Empezó como una página web sencilla. Terminó siendo una app instalada en mi celular. 🔥

## Instalación

1. Clona el repositorio
2. Abre el proyecto en **Android Studio**
3. Agrega tus GIFs en `app/src/main/assets/gifs/`
4. Dale ▶ Run
